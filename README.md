# `polar-arctic`
![Build and Test](https://github.com/BeaconBrigade/polar-arctic/actions/workflows/rust.yml/badge.svg)

`polar-arctic` is a crossplatform application for reading heart rate, ECG and acceleration data from the Polar H10.
GUI is made using [iced](https://github.com/iced-rs/iced), [plotters](https://github.com/plotters-rs/plotters) and 
[plotters-iced](https://github.com/Joylei/plotters-iced). The backend is built using [arctic](https://github.com/Roughsketch/arctic)
